# 🛒 Online Retail Sales Analysis using MySQL
## 📖 Project Overview
    The Online Retail Sales Analysis project is a beginner-friendly MySQL project designed to manage customers, products, and orders using a relational database. 
This project demonstrates essential SQL concepts and CRUD operations while analyzing retail sales data through various SQL queries.

## 🎯 Objectives
- Manage customer information
- Store product details
- Track customer orders
- Analyze retail sales data
- Perform sorting and filtering operations
- Learn relational database concepts using MySQL
